# Mr Ian Portfolio

## Overview

This project is a personal engineering and software development portfolio designed to present technical skills, projects, and professional interests.

The website incorporates a modern interface with interactive elements and simulated system behavior to reflect an engineering-focused development style.

---

## Features

### Boot System Simulation

* Terminal-style startup sequence
* Simulated system initialization process
* Automatic transition to the main interface

### Matrix Background Animation

* Canvas-based animated background
* Continuous rendering of falling character effects
* Performance-optimized implementation

### Interactive User Interface

* Cursor-based glow tracking
* Scroll-triggered animations
* Interaction-based visual feedback

### Project Display

* Interactive project cards
* Mouse-based 3D tilt effect
* Structured presentation of work categories

### Visual Design

* Dark theme interface
* Subtle neon accent styling
* Glassmorphism-based sections

## System Behavior

### Boot Sequence

The system simulates a startup process using scripted terminal output, followed by a controlled transition into the main interface.

### Background Rendering

The matrix animation is rendered using the HTML canvas element and continuously updated using timed intervals.

### Interaction Handling

User interactions such as scrolling, mouse movement, and clicks are used to trigger animations and visual responses.

## Setup and Deployment

### Local Use

1. Download or clone the repository
2. Open `index.html` in a web browser

### Deployment (GitHub Pages)

1. Upload project files to a repository
2. Enable GitHub Pages in repository settings
3. Access the site via the provided URL
